## Problem Statement
Urban areas suffer from parking shortages while private parking spaces remain unused.
ParkShare aims to connect drivers with available private parking spaces.

## Solution Overview
ParkShare is a full-stack web prototype that demonstrates:
- Role-based access (Driver / Owner)
- Parking discovery workflow
- Owner listing management
- Dashboard-based UI experience

## User Roles
### Driver
- Search for nearby parking
- View parking details

### Owner
- Create parking listings
- Manage listings from dashboard

## System Architecture
Frontend: React (Vercel)
Backend: Node.js + Express (Render)
Database: MySQL (Demo / simplified)

## Demo Mode Disclaimer
This project is a **functional prototype**.
Some backend logic is simplified to demonstrate workflows rather than production-grade scalability.
